# Análise de tempo de entrega de delivery

#### Autoria: Naira Gama

#### Objetivo:
Realização de uma análise exploratória dos dados, visualizando os fatores importantes para redução do tempo de entrega em operações de delivery.

#### Tecnologias utilizadas:
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn

#### Etapas:
- Carregamento automático dos dados
- Renomeação de colunas
- Pré-análise
- Limpeza dos dados
- Transformação dos dados
- Análise Exploratória dos Dados (EDA)
- Insights da análise
- Análise de interação entre fatores
- Dataset final para dashboard
